Content

This is the core of your website!

This involves a markup language known as HTML (Hyper Text Markup language)

All the text, images, videos are considered to be the "content"

Design

This is what makes your website look appealing to the audience.

It involves a styling language known as CSS (Cascading style sheets).

Things you can do with CSS:

  • position elements
  • adjust colors
  • Change text options (size,fonts, etc)

Bootstrap

There is a framework known as Bootstrap, a set of pre-programmed CSS file that contains almost all positioning and styling.

It is very popular for novice programmers due to its ease and flexibility.

Bootstrap offers a responsive CSS design, which provides support for all devices, such as phones, tablets, computers, and more.
