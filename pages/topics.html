9th Grade Topics

Scratch

Coding platform that utilizes an user- friendly interface, perfect for beginners.

  • Simple click and drag system
  • No spelling mistakes!
  • Lays the foundation for logical reasoning
  • Enables the user to think creatively

Robotics

You'll be using Lego Robots

Robotics is both an enjoyable and interactive way to introduce new students into software engineering.


  • Similar to scratch, easy to learn
  • Friendly interface
  • Interactive robot that can be controlled through basic pogramming

11th Grade Topics

Github

Github is an online hosting service that allows its users to share, collaborate, and remix projects.


Benefits of Github

  • Free online storage!
  • Can roll back to any savepoint
  • Can view/request edits for other people's projects

Bootstrap

There is a framework known as Bootstrap, a set of pre-programmed CSS code that contains almost all positioning and styling.


Bootstrap offers a responsive CSS design, which provides support for a variety of devices.


Highly recommended for novice programmers due to its ease and flexibility.

Coding languages:
